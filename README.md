# PICO Mug · Procedural Object Study

An interactive Three.js reconstruction of the PICO Industry Design Studio mug, built procedurally from two reference photographs.

## Preview

[Open the live Three.js experience](https://cnyaos.github.io/pico-mug-threejs/)

## Run locally

```bash
npm install
npm run dev
```

Drag to orbit, scroll to zoom, or use the controls for auto-rotation, reset, and an exploded structural view.
